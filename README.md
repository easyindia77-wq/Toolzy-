# 🚀 Toolzy.online

Free Ultra Fast PDF Tools Platform built with Next.js + Node.js

---

# 🌍 Live Website

Frontend:
https://toolzy.online

Backend API:
https://api.toolzy.online

# ✨ Features

✅ Compress PDF to 50KB  
✅ Merge PDF Files  
✅ Image to PDF  
✅ Resize PDF  
✅ PDF to Word

# ⚡ Tech Stack

## Frontend
- Next.js
- TypeScript
- CSS

## Backend
- Node.js
- Express.js
- Multer
- PDF-Lib
- PDFKit
- Ghostscript

# 📦 Installation

## 1. Clone Project

```bash
git clone https://github.com/yourusername/toolzy.git
```

---

## 2. Install Root Packages

```bash
npm install
```

---

## 3. Install Frontend Packages

```bash
cd frontend
npm install
```

---

## 4. Install Backend Packages

```bash
cd backend
npm install
```

---

# ▶️ Run Project

## Run Frontend + Backend Together

```bash
npm run dev
```
